Compiling a Simple Chat Client on Windows with SVN source (http://simplechatclien.sourceforge.net/)
The instructions are given the latest versions of programs that were available during the writing.

1.) Install Visual C++ Studio 2008
2.) Download and install Qt 4.6.3 for Windows (http://qt.nokia.com/downloads/windows-cpp-vs2008)
3.) Download and install CMake 2.8.2 (http://www.cmake.org/files/v2.8/cmake-2.8.2-win32-x86.exe)
    When installing, select "Add CMake to the system PATH for all users"
4.) Download QCA2 for Visual Studio and unzip to C:\qca2 (http://delta.affinix.com/download/qca/2.0/qca-2.0.1-vs2005.zip)
5.) Download and install Visual C++ 2008 Redistributable SP1 (http://www.microsoft.com/downloads/details.aspx?FamilyID=A5C84275-3B97-4AB7-A40D-3802B2AF5FC2&displaylang=en)
6.) Download and install Win32 OpenSSL v1.0.0a Light (http://www.slproweb.com/download/Win32OpenSSL_Light-1_0_0a.exe)
    When installing, select "Copy OpenSSL DLLs to: The OpenSSL binaries (/bin) directory"
7.) Download and install TortoiseSVN (http://downloads.sourceforge.net/tortoisesvn/TortoiseSVN-1.6.10.19898-win32-svn-1.6.12.msi?download)
8.) Create directory project_scc and in this folder create directory scc
9.) Execute SVN Checkout with TortoiseSVN in scc directory
    As the "Url of repository" enter: https://simplechatclien.svn.sourceforge.net/svnroot/simplechatclien/trunk
10.) Run the compile.bat file from the scc directory
11.) Run the scc.sln file from scc directory
12.) Click on ALL_BUILD and compile as a Release
